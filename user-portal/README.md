# پنل کاربر (HTML / CSS / JS)

رابط کاربری فارسی و راست‌به‌چپ برای خریداران: ورود با OTP، مشاهده درخواست‌های خرید، فرصت‌ها و ثبت درخواست جدید.

## پیش‌نیاز

- API بک‌اند در حال اجرا (پیش‌فرض: `http://127.0.0.1:8000`)
- حالت آزمایشی OTP فعال (`OTP_SANDBOX=true`) — کد ورود همیشه **11111**

## تنظیم آدرس API

فایل `js/config.js`:

```javascript
window.APP_CONFIG = {
  API_BASE: 'http://127.0.0.1:8000/api/v1',
};
```

## اجرا

### روش ۱ — از طریق بک‌اند

```bash
bash scripts/local-up.sh
```

باز کنید: http://127.0.0.1:8000/

### روش ۲ — سرور استاتیک

```bash
cd user-portal
python3 -m http.server 5500
```

باز کنید: http://127.0.0.1:5500/

## ورود (حالت آزمایشی)

1. شماره: `09121234567`
2. «دریافت کد تأیید»
3. کد: **11111**
4. داشبورد درخواست‌ها و فرصت‌ها

## ویژگی‌های UI

- زبان فارسی و چیدمان RTL
- فونت وزیرمتن
- اعداد و تاریخ با قالب `fa-IR`
- برچسب فرصت: بهترین / خوب / منصفانه
